Separate admin from strategic work.

  • No escalation triggers

Future: Separated Model
  • Offshore absorbs all admin (CAAs, med records)

  • 60-80 files per CE = higher intensity per case
  • Compensation aligned with throughput & value
  • Escalation criteria + milestone accountability
